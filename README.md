# GLC Svelte Scheduling System

A Svelte-based scheduling system for managing students, teachers, subjects, and rooms.

## Features

- Add and manage schedules
- Assign teachers and sub-teachers
- Bulk schedule creation
- Responsive design with TailwindCSS and DaisyUI
- Google Sheets integration

## Installation

```bash
git clone https://github.com/Vic0905/glcSvelte-schedulingSystem.git
cd glcSvelte-schedulingSystem
npm install
npm run dev
```
